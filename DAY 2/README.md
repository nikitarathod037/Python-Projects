# Day 2 – Pandas CSV Analysis

## 📌 Overview
A Python script that reads and analyzes student data from a CSV file using the Pandas library.

## 📁 Files
| File | Description |
|------|-------------|
| `day2.py` | Main Python script |
| `students.csv` | Input dataset with student records |

## 🚀 How to Run

1. Make sure you have Python and pandas installed:
```bash
   pip install pandas
```

2. Run the script:
```bash
   python day2.py
```

## 📦 Requirements
- Python 3.x
- pandas

## 📊 What It Does
- Reads student data from `students.csv`
- Performs basic data analysis using pandas
